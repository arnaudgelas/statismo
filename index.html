StatISMo - Statistical Image and Shape Models

A framework for building Statistical Image And Shape Models

Statismo provides a versatile framework for generating and manipulating Statistical Shape and Image Models. It supports all commonly known types of statistical models, including Shape models, Deformation Models and Appearance Models. The implementation and interpretation is based on Probabilistic PCA, which generalizes the standard PCA models and gives a fully probabilistic interpretation.

Dependencies

In order to use Statismo, a current version of HDF5 needs to be installed.

Install

Statismo is a header only library. No installation is required for the library itself. However HDF5 (CMake build) has to be installed. Depending on the type of model you build, you may also need install VTK or ITK .
