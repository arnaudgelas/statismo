StatISMo - Statistical Image and Shape Models

A framework for building Statistical Image And Shape Models

Statismo is a C++ library for generating and manipulating Statistical Shape and Image Models. It supports all commonly known types of statistical models, including Shape models, Deformation Models and Appearance Models. The implementation and interpretation is based on Probabilistic PCA, which generalizes the standard PCA models and gives a fully probabilistic interpretation.

Statismo has been tested on GNU/Linux and Windows. To simplify its use, Statismo provides Python bindings for the most commonly used functionality.

Dependencies

In order to use Statismo, a current version of HDF5 needs to be installed.

Install

Statismo is a header only library. No installation is required for the library itself. However HDF5 has to be installed, as well as any library that you want to use for representing shapes or images (such as e.g. VTK or ITK).
