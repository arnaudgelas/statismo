StatISMo - Statistical Image and Shape Models

A framework for building Statistical Image And Shape Models

Statismo is a C++ library for generating and manipulating PCA based statistical models. It supports all commonly known types of statistical models, including Shape models, Deformation Models and Image (intensity) Models. The implementation and interpretation is based on Probabilistic PCA, which generalizes the standard PCA models and gives a fully probabilistic interpretation.

Statismo has been tested on GNU/Linux and Windows. To simplify its use, Statismo provides Python bindings for the most commonly used functionality.

Dependencies

All the libraries on which Statismo depends are distributed together with the Statismo source. Depending on the type of models you want to build, you need in addition to install ITK or VTK.

Installation

Statismo is a header only library and requires, in principle, no installation. However, we provide a CMake based installation procedure, which builds the dependencies (HDF5), example programs and unit tests, and sets up a CMake configuration that makes it easy to use Statismo within your projects. Detailed build instructions can be found in the projects README.
