Detailed Description

template<class TRepresenter, class TScalarType, unsigned int TDimension>
class itk::StatisticalModelTransform< TRepresenter, TScalarType, TDimension >

An itk transform that allows for deformations defined by a given Statistical Model.

Statistical models (shape or deformation models) are often used to model the typical variations within an object class. The StatisticalModelTransform provides allows to use an itk::StatisticalModel via the itk::Transform interface. Thus, the itk::StatisticalModel can be used as a tranform in the standard itk registration framework.
